HOW TO RUN THE SERVER
IN FOLDER NewTallerRuby EXECUTE SERVER CONNECTION
IN TERMINAL PUTS ruby server.rb
THEN WE HAVE TU RUN THE CLIENT CONNECTION
IN TERMINAL PUTS ruby.rb
NOW YOU CAN PUTS THE COMMANDS.
THIS IS THE SYNTAX:
       get [key]  
       gets [key,key2] 
       set [key] [flags ] [expiration] [bytes] ENTER
	   [value]
       add [key] [flags ] [expiration] [bytes] ENTER
	   [value]
       replace [key] [flags ] [expiration] [bytes] ENTER
	   [value]
       append [key] [flags ] [expiration] [bytes] ENTER
	   [value_to_append]
       prepend [key] [flags ] [expiration] [bytes] ENTER
	   [value_to_prepend]
       cas [key] [flags ] [expiration] [bytes] [1] ENTER
	   [value]
	   
FINALLY TO RUN THE TEST, YOU HAVE TO PUT IN TERMINAL ruby test_spec.rb
